# ADRs — Amazing (decisões de arquitetura)

Este diretório contém os **Architecture Decision Records (ADRs)** do projeto.
Cada ADR registra **Contexto → Decisão → Consequências**.

> Regra: decisão importante que afeta estrutura, rotas, segurança, padrões ou evolução do sistema
> deve virar um ADR (mesmo curto).

## Índice

- ADR 0001 — Módulos escopados em `/s/{scope}` — **Aceito**
- ADR 0002 — Portal (Group) controla macro-acesso a módulos — **Aceito**
- ADR 0003 — Sidebar derivada de config (sem hardcode) — **Aceito**
